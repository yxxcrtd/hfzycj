智慧城市：“合肥城泊”APP上线 停车不再碰运气

发布日期:2015-01-26 来源:人民网

市民们平日里出门办事，最担心的可能就是要“抢车位”。如今，合肥的智慧城市建设给大家吃了一颗定心丸：“合肥城泊”手机APP正式上线，今后只要轻轻滑动手指，就可以帮您摆脱盲目找车位的问题，做到心中有“位”人不慌，不仅节省了时间，也避免了不必要的绕行。

人民网安徽频道从合肥市了解到，为了缓解城市停车压力，优化市区停车环境，方便广大市民出行，合肥城市泊车管理有限公司精心组织研发了城市路边停车位查询的“合肥城泊”手机APP，通过该款手机APP，市民可以快速查询到全市路边临时停车位的空闲数量、具体位置和收费标准等详细信息，并且可以通过路线导航快捷抵达目的地。目前只有安卓版本的“合肥城泊”APP ，ios版本的“合肥城泊”APP目前正在研发中，预计今年6月推出。通过“合肥城泊”不仅找车位省心了，将来随之改变的还有缴费方式的多样化，除了以前的现金、刷卡支付之外，待系统进一步完善之后，车主还可以通过预付卡、手机APP、微信、支付宝等方式缴费。

目前，通过“合肥城泊”APP可以查询到全市10000多个停车位的详细信息，包括交通枢纽、办公园区、医院、商场、游乐场所、景点等主要停车场所。其中，免费停车位7000多个，收费停车位3000多个，覆盖包河区、瑶海区、庐阳区、蜀山区、政务区等多个区域。而且，“合肥城泊”APP的上线，只是合肥启动 “城市智慧停车系统”项目的一个缩影。“城市智慧停车系统”将实现市区道路停车位智能化、规范化、有序化的日常管理，减少因盲目寻找停车位而产生的无效交通，优化配置停车资源，提高泊位的利用率并减少能耗。这套系统的研发使用具有很强的时代前瞻性，目前在国内属于领先水平；而系统的应用，不仅为广大市民出行停车排忧解难，同时也为城市的智慧管理收集积累了大数据，为政府部门相关决策提供重要依据。
