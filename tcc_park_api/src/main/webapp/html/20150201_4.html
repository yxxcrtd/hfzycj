合肥城市智慧停车系统发布

发布日期:2015-01-24 来源:人民网-安徽频道

昨日，市场星报记者从合肥城市智慧停车系统上线发布会上了解到，作为全国首个成熟的停车智能系统，合肥城市智慧停车系统正式上线，市民目前只要在安卓智能手机端下载手机APP，就能随时随地查询到附近路边停车位、收费标准等详细信息。

    

缘由：合肥115万辆车，5.85万个停车位

　　

负责该系统研发的合肥掌韵车聚信息科技有限公司总经理赵世介绍，合肥交警部门最新数据显示，目前合肥市机动车保有量已突破115万辆，每月的上牌量基本维持在1万辆左右，而目前合肥市区开放的停车场包括公益公用停车场、社会经营性停车场、道路临时停车场三大类。除了单位和小区自备专用停车场外，三类停车场总计约803处，5.85万个停车位。

　　

这也就是说，如果合肥115万辆车行驶在外，只有5.85万个停车位，停车难已成为社会问题。因此，他们公司研发了智能停车系统，现阶段开发的手机APP主要适用于安卓手机，IOS系统的手机APP预计在6月份可以上线。

　　

市民只要下载“合肥城泊”APP，就可以在地图上看到周边1000米的所有收费停车位。

    

现在：安卓手机下载，即时查询

　　

合肥城市泊车管理的合肥城市泊车管理有限公司执行董事应长春介绍，这是全国首个成熟的停车智能系统，车主可以通过停车诱导屏、手机APP等多种方式获取合肥市停车位的实时动态信息。

　　

譬如，如果你使用目前设置在解放电影院等5个地段的停车诱导屏，该诱导屏周边路段1分钟之前的所有空闲停车位，在哪个路段，你都可以看到。但是，如果该路段空闲停车位只剩下5个，诱导屏的显示系统将由绿色改变为红色预警，提醒你如果不抓紧时间，可能赶到该路段，空余停车位已被占据。

　　

而手机APP，可通过手机百度、360浏览器、UC浏览器等免费下载。下载后，可通过APP“应用分享”给好友。合肥执行公务的警车、消防车、救护车、工程抢险车、军车、残疾车免收停车费，新能源汽车根据政府文件2015年也免收停车费。

　　

未来：小区停车位可短时出租

　　

未来，城市智慧停车系统可以提供合肥统一的停车信息共享平台，借助这个平台可以建立合肥范围内的停车诱导系统，通过后台管理系统展示停车位动态数据，及时了解停车场剩余停车位状况。同时，该系统可以实现缴费方式多样化，除了以前的现金、刷卡，还可以通过预付费卡、手机APP、微信、支付宝等方式缴费。可能在今年春节之后，市民就可以通过网络支付的方式，直接在手机客户端缴纳停车费。

　　

并且，该智能停车系统也将升级，将社会停车位纳入其中，也就是说，如果小区内车位业主愿意的话，可以将自己的车位限时出租，其他有停车意愿的车主可以通过手机客户端在白天的某个时段租用小区内的停车位。
