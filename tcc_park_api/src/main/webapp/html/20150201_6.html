“合肥城泊”App上线 打开手机找停车位

发布日期:2015-01-24 来源:中安在线-安徽商报

昨日，合肥市城市泊车管理有限公司召开改制后的首个新闻发布会，“合肥城市智慧停车系统”正式上线。开发者称，这将是全国大中型城市中首个城市级别的“泊车云平台”。

　　

“合肥城泊”上线

如果有一位大神掌管合肥环城路之内停车位，那他一定是用掷骰子的方式决定停车位的归属。一款名为“合肥城泊”的App(目前仅开发出安卓版、IOS版本6月推出)正在抢走这位大神的骰子。

　　

合肥城市泊车管理有限公司执行董事应长春称，目前通过“合肥城泊”App可以查询到全市一万多个停车位的详细信息，包括交通枢纽、办公园区、医院、商场、游乐场所、景点等主要停车场所。其中，免费停车位7000多个，收费停车位3000多个，覆盖包河、瑶海、庐阳、蜀山等多个区域。

　　

停车费将可网上付

　　

与普通移动端软件不同，“合肥泊车”因其谋划者的背景而倍受关注。合肥城市泊车管理有限公司，前身为合肥市保安停车场经营有限公司，该公司掌管合肥市几乎所有的道路临时收费停车位。公开数据显示：城泊公司目前管理市区83个路段的3120个临时停车泊位。

　　

在应长春的概念里，合肥版的未来停车是这样的，App查找车位、通过诱导屏查看车位信息、Pos机打印驶入凭证、收费员打印驶出小票、通过移动互联网支付停车费。“支付方式可以是支付宝、微信支付，也可以是预付卡支付。”这意味着，在价格杠杆之外，合肥市对于“停车难”的治理将开始借力互联网思维。

　　

将建大型停车场

　　

“目前完成的仅仅是前四个部分。其中，停车诱导屏正在合肥市内五个点小范围试用。”应长春说。要完成整套系统，仅靠合肥城泊已经远远不够。此前，合肥掌韵车聚信息科技有限公司和通联支付网络服务股份有限公司也已加入到合肥版泊车系统的建造行列中。

　　

以目前的App为例，能够查询车位空闲的只有合肥城泊管理的3000个停车位，超过7000多的免费停车位只能查位置，不能查是否空置。“如果要实现免费停车位的空置查询，就需要为每个停车位装备无线地磁设备，一个无线地磁设备价值3000元，这是一笔巨大的投入。”

　　

另一方面，尽管云平台能够提升停车效率，但并未改变“车多位少”的供求格局。“要建新车位也需要钱，合肥城泊将尝试更多投融资途径来建设大型停车场。”

收费员打印驶出小票通过移动互联网支付停车费

　　

手机将对你的当前位置定位并搜索出你周边1000米以内所有街道的停车位空闲情况，包括总停车位、空停车位和该路段停车位的收费时间和收费标准，并且可以通过路线导航快捷抵达目的地。

　　

泊车管理方案包括七大部分，包括智慧泊车智能POS终端、智慧泊车收费管理系统、车位查询软件(App)、城市停车诱导发布系统、互联网产品缴费、无线地磁和预付费卡七大部分。
