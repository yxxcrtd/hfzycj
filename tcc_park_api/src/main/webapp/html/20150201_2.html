拿着手机找车位 市内停车不再难

—— “合肥城泊”智能停车APP上线 可实时查询空闲停车位

发布日期:2015-01-24 来源:中安在线

在合肥市的中心城区，停车位“一位难求”令众多车主头疼不已。以往，都是要靠碰运气才能找到车位，今后，找车位就可以不用再盲目兜圈，哪里有空位，手机上点一点就一目了然。记者获悉，一款名为“合肥城泊”的手机APP已宣布上线，这款应用软件，能实时查询到全市路边临时停车位的空闲数量、具体位置和收费标准，并且可以通过路线导航指引车主前往空闲停车位所在地，找车位不再难。

据悉，目前，在合肥市二环以内，约有2万个路边临时停车泊位。其中，收费的停车位共3120个，分布在市区83个路段，其中A级路段临时停车泊位516个，B级路段2604个。这些停车位均由合肥城市泊车管理有限公司(以下简称合肥城泊公司)统一收费管理。该公司已对这3000余个收费停车位完成了泊位划线、编号，并聘用了200余名收费人员使用移动智能POS机进行收费管理，实现了实时动态数据的及时上传和共享。正是基于这些有效数据，合肥城泊公司组织研发了“合肥城泊”手机APP。

　　

通过“合肥城泊”APP可以查询到全市1万多个停车位的详细信息，包括交通枢纽、办公园区、医院、商场、游乐场所、景点等主要停车场所。其中，免费停车位7000余个，收费停车位3000余个，覆盖包河区、瑶海区、庐阳区、蜀山区、政务区等多个区域。使用该款手机APP，只要轻轻滑动手指，就可以快速查询到全市路边临时停车位的空闲数量、具体位置和收费标准等详细信息，并且可以通过路线导航快捷抵达目的地。通过“合肥城泊”APP找车位，可以避免盲目绕行，为车主节省了大量时间。

　　

据悉，“合肥城泊”手机APP是一个开放的手机客户端平台，安卓手机用户可以通过手机百度、360浏览器、UC浏览器或在各类应用市场免费下载。目前，只推出了安卓版本的“合肥城泊”APP ，ios版本预计将在今年6月推出。

　　

另据了解，“合肥城泊”手机APP的上线只是合肥城市智慧停车系统的一个部分。今年1月，合肥城泊公司已在合肥宿州路与庐江路交口、解放电影院对面、花园街东侧等重要道路路口，设置了5处城市停车诱导屏，屏幕主要显示附近有停车位的路段名称、路线方向、实时空车位数量等，与手机APP上的数据信息保持同步，用以引导市民快速停车和疏导交通。

　　

“城市智慧停车系统”将实现市区道路停车位智能化、规范化、有序化的日常管理，减少因盲目寻找停车位而产生的无效交通，优化配置停车资源，提高泊位的利用率并减少能耗。这一系统的应用，不仅为广大市民出行停车排忧解难，同时也为城市的智慧管理收集积累了大数据，为政府部门相关决策提供重要依据。据悉，这套系统是全国首个成熟的停车智能系统，目前在国内属于领先水平。

　　

不仅找车位省心了，这一系统的投入使用，还将带来缴费方式的多样化。在合肥城泊收费人员使用的POS机上，除了现在可以使用的现金、刷卡支付方式之外，待系统进一步完善之后，车主还可以通过预付卡、手机APP、微信、支付宝等方式缴费。

　　

下一步，合肥市区将增设更多的城市停车诱导屏，“合肥城泊”手机APP也将进行进一步功能完善和软件升级，合肥市城市停车管理智能化将走向新阶段。(记者 宋艺)
