停车不再难：我市路边临时停车位可用手机查询！

2014-12-28 09:06

由合肥城市泊车管理有限公司（以下简称“合肥城泊公司”）组织研发的城市路边停车位查询手机APP“合肥城泊”今日正式上线，通过该款手机APP，市民可以快速查询到全市路边临时停车位的空闲数量、具体位置和收费标准等详细信息。“合肥城泊”APP的诞生标志着我市城市停车管理智能化走向新的阶段。

10000多个停车位纳入“合肥城泊”APP平台

目前，通过“合肥城泊”APP可以查询到全市10000多个停车位的详细信息，包括交通枢纽、办公园区、医院、商场、游乐场所、景点等主要停车场所。其中，免费停车位7000多个，收费停车位3000多个，覆盖包河区、瑶海区、庐阳区、蜀山区、政务区等多个区域。

“合肥城泊”APP提供的停车位数量将会逐渐增加、信息逐步扩大、功能逐步完善，进一步满足市民对全市停车位信息的需求。

可查询由“合肥城泊公司”管理的停车位实时信息

由“合肥城泊公司”统一收费管理的全市3120个停车位已经实现实时动态数据的及时上传和共享。市民通过“合肥城泊”APP可以及时了解这些停车位的信息，包括：空闲停车位的数量、停车位收费标准、收费时间、收费管理员等。

“合肥城泊公司”从2014年着手建设“城市智慧停车系统”，由该公司统一收费管理的停车位采用了移动互联网的管理方式，通过智能无线手持POS机，对全市路边停车位进行收费，从而实现停车位信息的真实有效、及时共享。

可查询全市免费停车位的静态信息

市民通过“合肥城泊”APP还可以查询全市免费停车位的静态信息，包括：免费停车位路段位置、停车位总数量、路线导航。

通过“合肥城泊”手机APP市民不仅可以及时有效的获取停车位信息，便捷的引导市民停车，提高停车效率；还在一定程度上，实现全市停车诱导，帮助缓解全市停车难题，改善城市交通拥堵现状。

市民当前可通过“360手机助手”、“豌豆荚”等安卓手机软件应用市场搜索到“合肥城泊”APP下载使用，ios版本的“合肥城泊”APP目前正在研发中。
