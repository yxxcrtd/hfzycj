合肥首个智能停车系统诞生 各类信息尽在“掌”控中

发布日期:2015-01-25 来源:合肥日报

在城市里驾车出行，让很多车主头疼的是，停车经常一“位”难求。不过，这一难题将能得到缓解。记者昨天从“合肥城市智慧停车系统”上线发布会上获悉，“合肥城泊”手机APP正式上线，通过它市民就能查询路边临时停车位的空闲数量、具体位置和收费标准，让出行更加智能和便捷。

“最智能”停车系统全国领先

目前合肥市机动车保有量已突破115万辆，每月上牌量基本维持在1万辆左右，停车位供需矛盾日益突出。

面对这一难题，“合肥城泊”手机APP应运而生。市民只要在手机上下载这一APP，就能快速查询到全市路边临时停车位的空闲数量、具体位置和收费标准等详细信息，并且可以通过路线导航快捷抵达目的地。

不仅找车位更加省心，车主的缴费方式也更加多样。待系统完善后，市民不仅可用现金和刷卡支付，还可体验手机APP、微信、支付宝等更多支付方式。

“这款系统也是全国首个成熟的停车智能系统，市民不仅能查询到路边停车位、收费标准等信息，还可通过‘停车诱导屏’快速找到停车位。”合肥城市泊车管理有限公司执行董事应长春介绍，收费用的POS机也是全国首个基于安卓系统的高端智能PDA POS机，可以称得上是目前最先进的POS机。

停车位信息尽在“掌”控中

目前，通过“合肥城泊”APP可以查询到全市1万多个停车位的详细信息，包括交通枢纽、办公园区、医院、商场、游乐场所、景点等主要停车场所。其中，免费停车位7000多个，收费停车位3000多个。由“合肥城泊公司”统一收费管理的3120个停车位已完成了泊位划线、编号，并使用移动智能POS机收费管理。

不仅是手机APP，合肥“城市智慧停车系统”的触角延展得更远。今年1月，全市已在宿州路、环城路和桐城路等路段的重要路口设置了5个“城市停车诱导屏”，主要显示附近路段的名称、行驶路线、实际车位状态等信息，指导市民快速停车、疏导交通。

据了解，更多的二级“城市停车诱导屏”将陆续增设，后期还将建立一级“城市停车诱导屏”，在市区主要交通干线上显示停车信息。应长春表示，城市智慧停车系统是合肥“智慧交通”的重要组成部分，目前还处于初步阶段，随着系统的不断完善，将让市民的出行变得更科技、更智能。

不过需要提醒市民注意的是，目前只有安卓版本的“合肥城泊”APP，ios版本的APP正在研发中，预计今年6月推出。
