合肥将建成全国首个“城市智慧停车系统”

2014-12-29 08:50

合肥城市泊车管理有限公司为缓解城市“停车难”，正加速上线“城市智慧停车系统”。即日起，市民可通过手机APP查询路边停车位实时信息，通过路边“停车诱导屏”快速找到停车位。至此，合肥将建成全国首个“城市智慧停车系统”。

由合肥城市泊车管理有限公司（以下简称“合肥城泊”公司）建设的“城市智慧停车系统”能够同时为市民和路边收费管理员服务。市民在智能手机端下载手机APP后，能够随时随地查询到附近路边停车位、收费标准等详细信息。路边停车收费管理员使用智能无线手持POS机，可对全市路边停车位进行管理，且智能POS机能自动计时、计费。同时，这套系统还可以对收费进行监督监管，并对收费数据、停车位实时信息、停车位利用率等数据进行分析，为城市管理提供依据。

从2015年元月，“合肥城泊”公司在合肥市重要交通路段已经建设了部分二级“城市停车诱导屏”。诱导屏设置在重要道路路口，主要显示附近路段的名称、行驶路线、实际车位状态等信息。通过“城市停车诱导屏”可以指导市民快速停车，疏导交通。

公司会陆续增设更多的二级“城市停车诱导屏”，并根据市民的具体需求，对手机APP做进一步功能完善和软件升级，更好的为市民服务，引导市民快速停车，缓解停车难题，为建设“智慧合肥”贡献力量。

“合肥城市泊车管理有限公司”的前身是“合肥市保安停车场经营服务公司”，属于国有独资企业，负责合肥市区道路临时停车泊位的管理和收费工作。为配合政府的城市管理，“合肥城泊”公司将按照相关部门的批准，扩大停车管理范围，今后还将参与“城市公共自行车管理”、“城市充电桩业务”等。
