停车神器智能APP“合肥城泊”亮相合肥

发布日期:2015-01-24 来源:中国网-中国视窗 记者：汪丽

你出行还在盲目地找空车位吗？你还在为找不到空车位而头疼吗？那么福利来了，日前，城市路边停车位查询系统“合肥城泊”手机APP上线，用户只要轻轻滑动手指，就可以摆脱盲目找车位的问题，不仅节省了时间，也避免了不必要的绕行。

“合肥城泊”让车主摆脱一位难求

为了缓解城市停车压力，方便广大市民出行，在合肥城市泊车管理有限公司（以下简称“合肥城泊公司”）精心组织研下，“合肥城泊”手机APP应运而生。通过该款手机APP，市民可以快速查询到全市路边临时停车位的空闲数量、具体位置和收费标准等详细信息，并且可以通过路线导航快捷抵达目的地。“合肥城泊”APP的推广使用对众多车主们来说无疑是个好消息，这也标志着合肥市城市停车管理智能化走向了新的阶段。

“合肥城泊”收费方式将多种多样

“合肥城泊”APP不仅能让车主更快捷方便地找到空车位，将来随之改变的还有缴费方式的多样化。除了以现金、刷卡支付之外，待系统进一步完善之后，车主还可以通过预付卡、手机APP、微信、支付宝等方式缴费。

目前，通过“合肥城泊”APP可以查询到全市10000多个停车位的详细信息，包括交通枢纽、办公园区、医院、商场、游乐场所、景点等主要停车场所。其中，免费停车位7000多个，收费停车位3000多个，覆盖包河区、瑶海区、庐阳区、蜀山区、政务区等多个区域。其中，由“合肥城泊公司”统一收费管理的全市3120个停车位已经完成了泊位划线、编号，并使用移动智能POS机收费管理，实现了实时动态数据的及时上传和共享，大大提高了停车效率，保障了交通秩序。

“合肥城泊”APP用户可免费下载

合肥城泊公司表示，软件的开发就是为了共享和惠民，因此“合肥城泊”APP是开放的手机客户端平台，市民可以通过手机百度、360浏览器、UC浏览器等免费下载，下载后，可通过APP“应用分享”分享给好友。不过值得注意的是，目前只有安卓版本的“合肥城泊”APP ，下一步，会根据市民的具体需求，对APP做进一步的功能完善和软件升级，同时，IOS版本的“合肥城泊”APP正在研发中，预计今年6月便可与广大车主见面。

“合肥城泊”将努力完善智慧停车

合肥城泊公司执行董事长应长春称，“合肥城泊”APP的上线，只是启动 “城市智慧停车系统”项目的一个缩影。据悉，今年一月合肥城泊公司在合肥市重要交通路段已经建设了部分二级“城市停车诱导屏”，旨在显示附近路段的名称、行驶路线、实际车位状态等信息，指导市民快速停车，疏导交通。下一步，公司会陆续增设更多的二级“城市停车诱导屏”，并增加停车泊位和公共停车场的开发建设，致力在合肥市打造一个智能、利民、有序、和谐的停车环境。
