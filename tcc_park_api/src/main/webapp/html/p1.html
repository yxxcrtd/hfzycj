“城市停车诱导系统”: 路边停车新指引 帮您迅速找车位

2014-12-25 08:59

当小汽车保有量越来越多时，停车难问题随之而来，在车位难求的老城区尤显突出。哪里还有泊位可以停车？今后会有停车诱导屏来帮忙！

由合肥城市泊车管理有限公司设立的首期“城市智能停车诱导系统”已经在合肥市的部分重要交通路段投入使用，近千个停车位数据会在诱导屏上实时更新。通过“城市智能停车诱导系统”，市民可以迅速了解各大停车路段的泊位情况，并根据路线提示快速找到可停靠泊位。

二级诱导屏帮您找车位

市区试点“城市智能停车诱导系统”，共设立5块诱导屏。覆盖范围主要以交通压力较大的一环区域为主，包括：寿春路、九狮桥街、长江中路、安庆路、花园街、淮河路、阜阳路、庐江路、红星路、宿州路、无为路、益民街、舒城路等多个停车路段的近千个停车位。

城市停车诱导屏分一级二级三级，目前设立的是二级诱导屏，设立在城市次干道上，可以看到：停车泊位路段名称、方位指示标志和停车泊位的空车位数量。今后，市民开车行驶在道路上，通过这些诱导屏可以及时了解各路段的停车位信息，高效、快速的选择最优停车位，节约停车时间，缓解城市交通拥堵。

“合肥城泊公司”正在加快建设“城市智能停车诱导系统”，根据其管理的泊车位范围，将逐步扩大停车诱导屏的覆盖面积。

停车诱导屏上发布的内容主要是面向动态行驶中的车辆，行驶中的车辆在可视距离内的停留时间很短。为了让驾驶员完整地读取整个版面信息，诱导屏会设定合理的信息发布间隔时间，有效分享泊车信息。
