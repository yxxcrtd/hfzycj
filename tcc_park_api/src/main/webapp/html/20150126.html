合肥市区至少有4个“黑停车场”哪些地方可临时停车？

发布时间：2015-01-26 07:50:08    来源：合肥网

1月23日，“合肥城市智慧停车系统”正式上线，以后去老城区之前，只要打开手机搜一搜，就能知道哪里可以停车。记者探访发现，省城市区至少有4个“黑停车场”。大家开车去市中心时，一定要长个心眼。

[调查]

哪些地方可临时停车？

1月24日，记者从合肥交警部门和市政部门获悉，老城区内基本上所有的正规临时停车泊位，均由上述两个部门划定，而在一些大型商场、银行、超市等门口划定的临时停车泊位，由于涉及慢车道及人行道，还需城管部门介入。

据了解，目前市区机动车道两侧的临时停车泊位共有3120个，由合肥城市泊车管理有限公司统一进行收费。而一些商场、银行门口的临时停车泊位，则由产权所属单位管理，通常不会收费，由城管部门负责监管，不能占据盲道或影响行人通行。有关人士表示，这些临时停车泊位主要分布在环城公园路以内及芜湖路和绩溪路。

部分路段存“黑停车位”

记者走访中发现，老城区少数路段还存有胡乱收费的“黑停车位”，这些所谓的“黑停车位”或原本就是免费的临时停车位，或是由私人随意在路面划定，然后供司机停车进行收费，收费价格也由收费人随意决定。

庐江路创元大厦对面人行道上，路南侧新划了一些免费停车位，然而22日上午记者开车前往探访，却发现有人在此收费，所穿服装与合肥城泊公司所属的收费人员服装十分相似，让人误以为是收费停车位。

九狮桥路与长江路交口东侧，有部分正规的收费临时停车泊位，然而就在这些车位的东侧人行道上，还有十几个停车位却是由私人所划，进行收费。步行街百盛广场门口有一个正规停车场，停车场外面是一块空地，这块空地也被人当成了“收费停车场”。徽州大道和环城南路交口东侧有一排停车位，原先是合肥城泊公司管理的临时收费停车位，后因故被取消。但是取消之后，很多车主不知情，就有人穿着之前收费人员所穿的老式衣帽，照常在此进行收费。

[提醒]

人是否在车内处罚不同

如果你到了老城区，还是分不清哪里可以允许临时停车，从而出现了违停，如果当时你在车上，那么也不要太担心。交警部门表示，车辆违停驾驶人在车内的情况下，一般会给予口头警告，要求其立即开走，而不会罚款或扣分。但是如果驾驶人不在车内，或者是拒绝离开，交警部门就要“贴单”或是直接将车拖离现场了，“处罚一般是记3分罚200元，违法情节较轻的，罚100元不记分。”

遇“黑收费”可报警处理

目前“合肥城市智慧停车系统”已正式上线，有关负责人表示，车主驶入收费泊车位停车，应该主动向收费员素要驶入凭证，驶出时，应向收费员索取收费发票。这样如果车辆出现刮擦等问题时，公司可以及时调查解决。当车主离开时，一定要及时联系收费员，否则系统会自动续费，给车主造成不必要的损失。“如果碰到黑收费，车主可以有权拒绝缴费，并报警处理。”工作人员提醒说，目前公司还对所有收费路段停车位进行了统一编号，广大司机在停车前一定要分辨正规停车位和“黑停车位”，以免利益受损。

黑车位位置

1.庐江路创元大厦对面人行道上路南侧(应免费)
　　2.九狮桥路与长江路交口东侧人行道(应禁停)
　　3.步行街百盛广场门口有一个正规停车场外面空地(应禁停)
　　4.徽州大道和环城南路交口东侧一排停车位(应禁停)

看标识

正规停车位一般宽为2.2米至2.5米，长度5米左右，车位线清晰笔直，拐角处多有重叠痕迹。除此之外，停车位附近会设有明显的临时停车标识。

看着装

合肥城泊公司工作人员表示，正规收费人员均着印有“合肥城泊”字样的绿色马甲、红色帽子、红色收费包，正规收费的发票是由合肥市税务局统一印刷，印有“合肥城市泊车管理有限公司”的POS机机打专用发票。

看时间

同时在每个正规收费路段，都设有合肥城市泊车管理有限公司临时停车收费标牌；正规收费时间为A级区域部分路段收费时间：7：30-21：30，部分路段收费时间：7：30-19：00，B级区域收费时间：7：30-19：00。
