“合肥城泊”APP上线 找车位“神器”可查2万多停车位

发布日期:2015-01-24 来源:合肥在线-江淮晨报

核心提示： “合肥城市智慧停车系统”已经上线，市民可通过手机APP查询到市区近3000个临时停车位的实时信息，以及20000多个停车位的分布情况。

   

开车去闹市区，找车位可能算得上是头号困扰了。昨天，记者从合肥市国资委召开的新闻发布会上获悉，“合肥城市智慧停车系统”已经上线，市民可通过手机APP查询到市区近3000个临时停车位的实时信息，以及20000多个停车位的分布情况。

    

找车位有“神器”区分收费免费

   

合肥城市泊车管理有限公司(以下简称合肥城泊公司)近日推出的“合肥城泊”手机APP是针对停车位难找的困扰，专门开发的一款软件。“通过APP，市民可以快速查询到路边临时停车位的具体位置和收费标准。”合肥城泊公司执行董事应长春告诉记者，目前全市有划线车位5万多个，其中位于主城区的2万多个车位信息，可以在这款APP上查到，“这些停车位分布在交通枢纽、医院、游乐场所、景点等，还有大量的路边停车位。”应长春说。

   

记者看到，在APP上，免费停车位用绿色图标做出标记，而蓝色标记代表收费停车位。对于市民来说，除了找到停车位和了解收费标准之外，最为关心的，还是能不能快速“锁定”空余车位。应长春表示，目前，有近3000个收费停车点，已经实现实时数据上网，在APP上，市民可以看到这些收费停车点是否空闲。其余车位，暂时只能显示地理位置和收费情况等信息。

   

“因为这些车位主要集中在露天场所，只能通过人工收集或者地磁收集，所以无法短时间内全部实现实时数据上传。”应长春告诉记者，“APP上标记的2万多个停车位，以免费为主。再加上城市道路建设、改动的因素，在大量的路边停车位下安装地磁装置，短时间没有办法实现。”

    

未来停车付费方式更多样

   

应长春表示，这一次除了实现“找车位”的功能之外，合肥城泊还联合通联支付公司，待系统完善之后，车主还可以通过预付卡、手机APP、微信、支付宝等手段缴费。

   

“收费的近3000个停车位，我们已经完成划线和编码，可以使用通联的POS机收费管理，小票上还能显示历史欠费情况。经常有市民抱怨遇到非法划线收费无法识别，通过这项业务，也可以区别开来。”应长春说。

   

事实上，近3000个停车位的实时信息，尚不能满足市民出行的需求。不过，应长春在接受采访时表示，未来，这款APP也将面对社会经营性停车场、酒店、商场等停车位信息开放，实现数据联网。届时，市民可以通过这款APP了解到更全面的停车位信息。

   

目前，“合肥城泊”安卓版已经上线，IOS版本预计在今年6月推出。

市中心5处停车位 装上了诱导屏

   

昨天，记者从合肥市国资委召开的发布会上获悉，今年1月，5处市区重要交通路段建设了二级城市停车诱导屏，主要显示附近路段的名称、行驶路线、实际车位状态等信息，可以引导市民快速停车、疏导交通。

   

据介绍，这5处诱导屏分别位于宿州路解放电影院门口、宿州路与庐江路交口、长江中路相王大酒店门口、花园街东侧、益民街与桐城路交口。
