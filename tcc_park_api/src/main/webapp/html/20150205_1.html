人代会关注停车管理 路侧停车或将立法

2015年02月02日 09:52:26
来源：新京报作者：黄颖 郭超 张媛

北京市第十四届人代表会三次会议昨日闭幕，据市人大议案审查委员会介绍，今年共收到191件代表议案，集中关注停车管理和缓解拥堵等4方面的问题，有8件代表提出的关于停车管理方面的议案，被合并为一项议案。

议案审查委员会分析，代表意见最集中的是停车管理问题，建议优化停车设施规划布局，加强停车设施建设，规范停车管理。

另据了解，在人大常委会工作报告中，有关2015年立法工作的章节也提到，要对机动车停车管理规定，在预案研究基础上开展立项论证。

此前，北京市交通委主任周正宇也曾表示，今年将下大力整治违法停车，力争年内制定综合性的措施，最为混乱的路侧停车管理将尽快通过立法解决，目前已经启动立法的前期调研工作。

据了解，在北京市人大常委会工作报告中也提到，今年拟对机动车停车管理规定在进行预案研究基础上开展立项论证。周正宇告诉记者，目前，他得知北京市人大已经加快推进工作，最快明年有望实现停车管理的立法。

此次大会共收到代表提出的议案191件中，23件合并为加强机动车停车管理、提供农村医疗卫生服务水平等四项议案交由市政府办理。

问题1

停车场收费监管无序

人大代表、北京律师协会副会长高子程表示，目前停车场的监管和收费基本上处于无序和乱象状态，街道物业圈地乱收费，行车道、消防通道变成停车场，车辆开进绿化带，黑停车场划线收费，很多正规停车场也私划车位牟利，路边停车位未备案，停车员不给足额发票，超几小时后漫天要价，加剧了交通拥堵、治安隐患、公共资源私有化等问题。且有走向更加危险的倾向，有必要采取统一监管政策措施加以治理。

“我曾到交通委进行调研，但交通委将这个问题交到了街道办事处。”他称，主要问题是执法主体不明确，停车场管理涉及市政行政管理部门、规划行政管理部门、价格行政管理部门等众多部门，却没有一个专门的管理部门。另外就是可以作为停车场的公共资源界定不明，加上相关部门重视不够，利益多元、停车场权属多元，执法依据和执法力度不够。

【回应】

推动立法解决停车管理乱象

对于这一问题，高子程表示，建议城管执法局统一监管停车场秩序，政府制定统一的停车场权属界定标准并划定区域，政府制定统一的招投标政策或承包政策，按市场规则利用公共停车场区域。根据政策聘用相关专家团队，根据不同区域、不同时段，明确各类公共场地停车场收费标准。

“去年已经下社区调查停车问题，要摸清底数和需求。”交通委主任周正宇介绍，今年将下大力整治违法停车，力争年内制定综合性的措施，最为混乱的路侧停车管理将尽快通过立法解决。

周正宇表示，北京停车问题涉及方面多，不仅是违法停车问题，停车供需矛盾突出，引起很多停车乱象。根本解决还需要立法来推进，已经启动立法的前期调研工作。

在北京市人大常委会工作报告中也提到，今年拟对机动车停车管理规定在进行预案研究基础上开展立项论证。周正宇告诉记者，目前，他得知北京市人大已经加快推进工作，最快明年有望实现停车管理的立法。

问题2

停车位缺口281万个

安丽娟代表介绍，目前北京汽车保有量已达到561万辆，但车位只有280万个，缺口281万个。此外，北京市机动车数量还在持续增长。

市人大代表马小兰也提出，当前新建小区机动车停车位配建比例低。目前，北京停车位配建指标是三环内0.3个/户，三环外0.5个/户的配建标准，与全市机动车拥有率相比，缺口较大。

导致停车难还有小区周边乱停车、市区政府行政单位、学校、医院、公园景区、商业购物中心等场所没有停车场，或车位缺乏的问题。安丽娟分析，这一情况导致来往车辆占用公共道路停车，进而引发车主乱停乱放行为，反而加剧了停车难。

此外，目前一些小胡同，原本还可以走路、骑自行车，但目前往往道路一侧或两侧都停了机动车，导致居民出行非常困难。

【建议】

配建公共车场统一收费标准

安丽娟建议，为解决停车难应从车辆增长速度的源头管控，虽然目前北京市已经采取了摇号、管控外地进京车辆、增加公交线路辐射各区点位等措施，已经收到了良好效果，但未来应管控同步实施，控制更加严格。

例如，买车除了从资格、摇号等进行限制，更要从有无合法停车泊位上控制车辆增长，以解决居民汽车无处可停的问题。

安丽娟建议，政府要出台相关鼓励政策，引导和鼓励社会力量参与公共停车场的建设，把停车建设管理作为扶持产业推动和发展，逐渐将停车场的建设、维修、服务等管理运行形成一套完整的经营产业链，按照规定的比例配建公共停车场，确定管辖权和统一的收费标准。

此外，还要逐步完善停车法律，可对有条件的繁华大街在不影响交通的前提下，适当划设一些停车位及设立准许停车的标志，解决购物、出行停留和工作场所周围停车难的矛盾。

问题3

新小区车位价高难求

安丽娟代表表示，老旧小区车位尤其紧张，见缝停车隐患较多。她称，受历史规划所限，老旧小区停车位较少，随意停车导致地锁成了“装饰物”，南二环的杨家园小区就有200多个各种地锁。

“晚上觉都睡不好。”安丽娟说，许多车主都会因车辆被剐蹭、扎胎放气等隐患担惊受怕。

马小兰代表也提出，老旧小区规划车位较少甚至未规划机动车停车位，小区内乱停乱放，影响市民正常生活，甚至堵塞了消防通道。

安丽娟介绍，除了老旧小区因历史规划问题车位少之外，新建小区停车位也价高而难求。

“公租房每10户一个车位。”她说，普通小区每3户一个车位，且车位价格均在10万到14万元之间，让住户难以承受，小区免费停车空间越来越小，住户多而车位少的问题广泛存在。

【建议】

小区周边单位应开放停车场

对于老旧小区因历史规划缺乏停车位的问题，安丽娟认为，政府要支持社会力量在老旧小区建设“上天入地”式的停车场。

她举例说，传统停车位50辆需要1650平方米，而立体停车位50辆只需50平方米，并更加节省资金。

此外，还可以在小区周边道路上划线车位实行限时收费停车，小区周边单位向居民开放停车，实行限时收费等。

马小兰则建议，增建停车位是缓解老旧小区停车问题的重要措施。利用空闲地、边角地进行停车位改造。安丽娟代表建议，新建小区要按照北京市宜居和谐的规划和标准规划车位数量，排查所有停车设施并纳入公共停车场，高度挖掘周边、立交桥等停车资源。

此外，物业公司也可以协商集资，发动社会力量建设停车场，完善相关配套设施的服务。
