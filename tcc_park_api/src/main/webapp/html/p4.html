在合肥，未来理想中的路边停车是什么样？

2014-12-23 07:30

未来，合肥市民是这样实现路边临时停车的：

出行前，市民能够从手机APP、城市停车诱导屏、广播中获得附近停车泊位的信息和剩余车位信息。车辆驶入路边临时停车位时，有无线地磁感应车辆的驶入时间，并通过手机APP或短信通知车主车辆驶入的泊位号、驶入时间、地点、收费标准等详细信息。

当车辆驶出时，无线地磁感应车辆驶出信息、计算停车时间，按照收费标准自动结算停车费用，系统自动发送通知短信。车主可以选择运用预付费卡、银行卡以及支付宝、微信等多种付费方式，快速实现停车费的缴纳……

把车位探测、停车收费、停车诱导、移动支付等技术应用于城市智慧停车，并运用移动互联网、物联网等多种方式把全市所有泊位资源全部共享。

当下，为了缓解城市停车压力，优化市区停车环境，保障交通秩序，合肥城市泊车管理有限公司启动了“城市智慧停车系统”项目。目前，由“合肥城泊公司”统一管理的停车位已经完成了泊位划线、编号，并使用移动智能POS机收费管理，在合肥市重要交通路段设立了城市二级诱导屏，组织研发了“合肥城泊”APP用于查询全市停车位信息。

“城市智慧停车系统”将实现市区道路停车位智能化、规范化、有序化的日常管理，减少因盲目寻找停车位而产生的无效交通，优化配置停车资源，提高泊位的利用率并减少能耗。

在这个智能化、信息化、数字化的时代，“合肥城泊公司”憧憬的最终目标和愿景或许就在不久的将来逐步落实、普及。
