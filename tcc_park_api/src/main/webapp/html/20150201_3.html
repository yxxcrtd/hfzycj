全市万余停车位尽在“掌”握

发布日期:2015-01-24 来源:合肥晚报

周末想去省立医院看病，又担心庐江路上没地儿停车，出门前透过手机APP就能查询到路边停车位的实时信息。这不是科幻大片，而是从今年1月悄然上线的“合肥城泊”APP。昨日记者从上线发布会获悉，合肥将成为全国首个建成“智慧停车系统”的城市。

　　

智慧停车合肥赶超北上广

　　

在堵车严重的宿州路上，一辆私家车刚刚停入车位，一分钟后，在“合肥城泊”APP上系统中该区域的停车位数就会自动扣除该“空位”，并且将车主导航到有空位的准确位置。

　　

而现在这款停车神器“合肥城泊”安卓版的APP已经可以通过“360手机助手”、“豌豆荚”等安卓手机软件应用市场搜索到“合肥城泊”APP下载使用。市民可通过“合肥城泊”APP及时了解停车位的信息，包括：空闲停车位的数量、停车位收费标准、收费时间、收费管理员等。苹果ios版的“合肥城泊”APP将稍迟一步，预计会在今年上半年面世。

　　

昨日，记者从“合肥智慧停车系统”上线发布会了解到，该系统由合肥市国资委下属企业——合肥城市泊车管理有限公司建设，其初衷是缓解城市“停车难”。该系统从技术上已经赶超北上广，而合肥也将凭此成为全国首个建成“智慧停车系统”的城市。

　　

目前，通过“合肥城泊”APP可以查询到全市10000多个停车位的详细信息，包括交通枢纽、办公园区、医院、商场、游乐场所、景点等主要停车场所。其中，免费停车位7000多个，收费停车位3000多个，覆盖包河区、瑶海区、庐阳区、蜀山区、政务区等多个区域。

　　

掌上可查二环内空车位

　　

这款停车神器为啥能一手掌握车位实时信息，奥秘何在？据研发公司总经理赵世透露，“合肥城泊”APP之所以能实时显示停车位信息，奥妙就在于路边车位管理员手中的智能POS机。它们会将实时的数据传回“停车动态数据处理中心”，并且同步到“智慧停车云平台”上。

　　

不过美中不足的是，目前，该系统只能显示二环内收费车位的实时信息。由“合肥城泊公司”统一收费管理的全市3120个停车位已经实现实时动态数据的及时上传和共享。今后，随着APP版本的不断升级，“合肥城泊”APP可供查询的停车位范围也将由二环扩展至全市。

　　

据研发公司总经理赵世介绍，“合肥城泊”APP仅仅是“合肥智慧停车系统”七大功能中的一部分。系统目前已投入使用的还包括“智慧泊车智能POS终端”、“智慧泊车收费管理系统”、“城市停车诱导发布系统”等。

　　

新年伊始，细心的市民可能已经发现在宿州路解放电影院对面、桐城路益民街等重要交通路段，竖立起了一面特殊电子显示屏，上面标记着周边的车位信息。赵世在发布会上透露，这些设置在重要道路路口的“城市停车诱导屏”将显示出附近路段的名称、行驶路线、实际车位状态等信息。“当某一路段车位数不足5个时，屏幕会自动显示为红色，提醒车主择路停车。”

　　

上班后小区车位将能收租子

　　

截至2014年6月，合肥市机动车保有量已达到113万辆，其中市区机动车就有70万辆，并且仍在以年均20%的速度递增。

　　

合肥城市泊车管理有限公司执行董事长应长春表示，在调查中他们发现，除单位和小区自备专用停车场外，市区共有停车场803处，共计5.85万个泊位。但由于路边车位未统一管理，缺乏停车诱导系统及查询工具，“停车难”、“乱停车”等城市病困扰着“有车一族”。

　　

昨日记者从发布会上获悉，待“合肥城泊”APP退出后，后续还将启用无线地磁、预付费卡，以及微信、支付宝等相关互联网产品缴费等三大功能。这也意味着停车位的实时信息有望扩展到免费车位，并且实现支付宝、微信支付，全城通用停车卡等多功能。

“以后白天车主去上班，还可以把小区自家的空余停车位发布到智慧停车系统里，周围临时办事找不到车位的人可以租用。车主一边上班，一边就可以轻松收收租子。”赵世还透露，今后，已建成的“智慧停车系统”还将对停车信息全方位的采集，形成泊车大数据，为城市规划提供基础数据，同时为交通和城市管理部门提供决策依据。
